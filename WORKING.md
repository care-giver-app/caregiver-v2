Requirements:

- User can add one or more tracking events at a time
- User can view a daily timeline of the events on a particular day
- User can view the last logged event and it's data for each tracker on a single view
- User can view upcoming and past appointments
- User receives some in app indication for appoinments coming up within 2 weeks
- User can see trend data for events for each tracker (number of times logged per time frame)
- User can see a scatter plot where hour of day is y axis and date is x axis
- User can adjust time frame for any/all charts or stats
- User can see graphs for numerical trackers for value vs time
- User should have the ability to quick log events that require no data
- User should see other care givers in their group
- User should see who is the active care receiver that is selected
- User should be able to change the care receiver
- User should be able to invite other care giver
- User should be able to create a tracker for a care receiver
- User should be able to see all event data for a specific event
- User should be able to edit and delete events
